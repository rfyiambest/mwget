## mwget 

这是一个简单实现的多进程下载工具

### 介绍

安装
```
pip install mwget
```

使用mwget下载
```
python -m mwget.multiwget http://mirrors.163.com/centos/7/isos/x86_64/CentOS-7-x86_64-NetInstall-1611.iso
```
